# Keybinding Referenz für The Whispering Dark
Diese Liste enthält alle gültigen Strings für die `keybindings.json`. 
Die Engine nutzt `winit` KeyCodes via `serde`.

## Buchstaben
Wichtig: Alle Buchstaben müssen das Präfix **'Key'** haben.
- `KeyA`, `KeyB`, `KeyC`, `KeyD`, `KeyE`, `KeyF`, `KeyG`, `KeyH`, `KeyI`, `KeyJ`, 
  `KeyK`, `KeyL`, `KeyM`, `KeyN`, `KeyO`, `KeyP`, `KeyQ`, `KeyR`, `KeyS`, `KeyT`, 
  `KeyU`, `KeyV`, `KeyW`, `KeyX`, `KeyY`, `KeyZ`

## Zahlen (Hauptreihe)
Wichtig: Alle Zahlen müssen das Präfix **'Digit'** haben.
- `Digit1`, `Digit2`, `Digit3`, `Digit4`, `Digit5`, 
  `Digit6`, `Digit7`, `Digit8`, `Digit9`, `Digit0`

## System- & Sondertasten
Diese werden meist ohne Präfix geschrieben:
- `Space` (Leertaste)
- `Escape`
- `Enter`
- `Tab`
- `Backspace`
- `CapsLock`
- `Delete`
- `Insert`
- `Home` / `End`
- `PageUp` / `PageDown`

## Pfeiltasten
- `ArrowUp`
- `ArrowDown`
- `ArrowLeft`
- `ArrowRight`

## Modifikatoren
- `ShiftLeft`, `ShiftRight`
- `ControlLeft`, `ControlRight`
- `AltLeft`, `AltRight`

## Nummernblock (Numpad)
- `Numpad0` bis `Numpad9`
- `NumpadAdd`, `NumpadSubtract`, `NumpadMultiply`, `NumpadDivide`
- `NumpadEnter`, `NumpadDecimal`

---
*Hinweis: Falls eine Taste im Log als 'Unbekannt' erscheint, prüfe die Groß-/Kleinschreibung. Die Namen müssen exakt so geschrieben werden wie oben gelistet.*
